Configuration technique

Connectez le stockage JSONBin, le formulaire de contact EmailJS, le référencement et la sécurité du backoffice.

⚠️ Ce site n'a pas de serveur : les clés saisies ci-dessous restent dans le navigateur (localStorage) et transitent dans le code livré au client. Utilisez une clé JSONBin restreinte (lecture/écriture sur ce bin uniquement), pas la clé principale de votre compte. Cette protection est dissuasive, pas une sécurité serveur.

JSONBin — stockage du contenu

Créez un compte gratuit sur jsonbin.io, un bin contenant {{ '{}' }}, puis une clé d'accès en lecture/écriture limitée à ce bin.
